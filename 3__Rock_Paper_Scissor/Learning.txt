============================================================
ASCII ART & COLOR IN C PROGRAMMING
============================================================

ASCII art is a technique of creating visual designs using
standard keyboard characters. In C programming, ASCII art
is widely used in console applications to improve user
experience without using graphics libraries.

------------------------------------------------------------
1. WHAT IS ASCII ART
------------------------------------------------------------

ASCII Art is made using characters such as:

#  *  @  $  %  &  |  _  -  /  \  ()

Example (Simple Box):

##########
#  HELLO #
##########

ASCII art works because every character occupies a fixed
space in the console (monospace font).

------------------------------------------------------------
2. WHY ASCII ART IS USED
------------------------------------------------------------

• Makes console programs visually attractive  
• No need for graphics libraries  
• Works on low-end systems  
• Useful for banners, menus, games, loaders  
• Common in CLI tools, games, and system programs  

------------------------------------------------------------
3. ASCII ART IN C (BASIC EXAMPLE)
------------------------------------------------------------

printf("###########\n");
printf("#  WELCOME #\n");
printf("###########\n");

Key point:
ASCII art is just text printed using printf().

------------------------------------------------------------
4. ADDING COLOR TO ASCII ART
------------------------------------------------------------

By default, console text is white.
Colors are added using ANSI escape codes.

Syntax:
\033[<color_code>m

Reset color:
\033[0m

------------------------------------------------------------
5. ANSI COLOR CODES
------------------------------------------------------------

Text Colors:
30 - Black
31 - Red
32 - Green
33 - Yellow
34 - Blue
35 - Magenta
36 - Cyan
37 - White

Example:
\033[31m  -> Red
\033[32m  -> Green
\033[0m   -> Reset

------------------------------------------------------------
6. COLORED ASCII ART EXAMPLE
------------------------------------------------------------

printf("\033[33m");
printf("   _______\n");
printf("  |  CUP  |\n");
printf("  |_______|\n");
printf("\033[0m");

This prints the ASCII art in yellow color.

------------------------------------------------------------
7. MULTI-COLOR ASCII ART
------------------------------------------------------------

printf("\033[31m###\n");   // Red
printf("\033[32m###\n");   // Green
printf("\033[34m###\n");   // Blue
printf("\033[0m");         // Reset

You can mix colors line by line or section by section.

------------------------------------------------------------
8. PLATFORM COMPATIBILITY (IMPORTANT)
------------------------------------------------------------

• Linux / macOS:
  ANSI colors work by default

• Windows:
  Older CMD does NOT support ANSI by default
  Windows 10+ supports ANSI in most terminals

Alternative (Windows):
Use SetConsoleTextAttribute() from windows.h

Beep Sound in C (Windows)

--------------------------------------------------
Basic Example (ASCII Bell)
--------------------------------------------------

#include <stdio.h>

int main() {
    printf("\a");   // ASCII Bell character
    return 0;
}

If sound is enabled in the terminal, this will produce a short beep.
This method is part of the C standard.


--------------------------------------------------
Using Beep in Practical Programs
--------------------------------------------------

Example: Timer Completion Alert

printf("\nTIME'S UP!\a\n");

This approach is commonly used in:
- Countdown timers
- Stopwatches
- Alarm programs
- Error or warning notifications


--------------------------------------------------
Windows-Specific Beep (More Reliable)
--------------------------------------------------

Windows provides a dedicated API for sound output.

Using Beep() (Windows only):

#include <windows.h>

int main() {
    Beep(750, 300);   // Frequency in Hz, Duration in milliseconds
    return 0;
}


--------------------------------------------------
Advantages of Beep()
--------------------------------------------------

- Custom frequency control
- Custom duration control
- More reliable than ASCII Bell
- Does not depend on terminal sound settings


--------------------------------------------------
Difference Between \\a and Beep()
--------------------------------------------------

ASCII Bell (\\a)
- Portable (C standard)
- Depends on terminal settings
- May be silent in some environments

Beep()
- Windows-only
- Guaranteed sound output
- Full control over tone and duration


--------------------------------------------------
Conclusion
--------------------------------------------------

Use \\a for simple, portable alerts.
Use Beep() when working on Windows and reliable sound feedback is required.
